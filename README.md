# 🍔 Sistema de Gerenciamento - Hamburgueria

Este é um projeto em **Python** desenvolvido para simular o funcionamento de uma hamburgueria, utilizando o conceito de **CRUD (Create, Read, Update, Delete)**.

O sistema permite gerenciar clientes, visualizar informações e servir como base para expansão com pedidos, pagamentos e entregas.

---

## 🚀 Funcionalidades

O sistema possui um menu interativo com as seguintes opções:

1. **Cadastro de Cliente (Create):**
   Permite registrar novos clientes com nome e telefone.

2. **Listagem de Clientes (Read):**
   Exibe todos os clientes cadastrados no sistema.

3. **Cardápio (Em desenvolvimento):**
   Estrutura para exibir produtos disponíveis (hambúrgueres, bebidas, etc.).

4. **Solicitação de Pedido (Em desenvolvimento):**
   Permitirá ao cliente escolher itens do cardápio.

5. **Pagamento (Em desenvolvimento):**
   Simulação de pagamento com cálculo de valores.

6. **Acompanhamento de Entrega (Em desenvolvimento):**
   Status do pedido (preparando, saiu para entrega, entregue).

7. **Cancelamento de Pedido (Em desenvolvimento):**
   Opção para cancelar pedidos realizados.

---

## 🛠️ Tecnologias e Estruturas Utilizadas

O projeto foi desenvolvido com conceitos básicos de Python:

* **Listas (`[]`)** → Armazenamento de clientes
* **Dicionários (`{}`)** → Representação dos dados dos clientes
* **Laço `while`** → Menu interativo contínuo
* **Condicionais (`if/elif/else`)** → Controle de fluxo
